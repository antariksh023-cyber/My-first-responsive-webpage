Responsive Webpage

A modern, fully responsive webpage built using HTML5 and CSS3. This project was created to practice responsive web design by applying concepts like Semantic HTML, CSS Flexbox, Media Queries, and relative CSS units.

🚀 Features

* Fully responsive layout
* Mobile, tablet, and desktop support
* Semantic HTML structure
* CSS Flexbox for layout
* Media Queries for different screen sizes
* Clean and organized code
* Modern user interface

---

🛠️ Technologies Used

* HTML5
* CSS3

Concepts Practiced

* Semantic HTML
* CSS Flexbox
* Media Queries
* Relative CSS Units (`rem`, `%`, `vw`, `vh`)
* Responsive Web Design
